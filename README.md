# CrystalOfKnowledge
こちらは読書記録管理システムです。
日ごろの読書記録を付け、書籍の管理・感想のまとめを行い、登録ユーザ間で読書記録の共有やモチベーションの維持を目的としている。
ユーザは複数人を想定。
- 新規登録 メール認証→ユーザ名・パスワードの登録を行う。
- ログイン
このソフトはどんなもので、何ができるのかを書く
